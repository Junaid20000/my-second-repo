    • Span Tag

    The span> tag is an inline container used to mark up a part of a text, or a part of a document. The span> tag is easily styled by CSS or manipulated with JavaScript using the class or id attribute. The span> tag is much like the div> element, but div> is a block-level element and span> is an inline element.

    DIV Tag

    The DIV tag defines a division or a section in an HTML document. The DIV tag is used as a container for HTML elements - which is then styled with CSS or manipulated with JavaScript. The DIV tag is easily styled by using the class or id attribute. Any sort of content can be put inside the DIV tag! Note: By default, browsers always place a line break before and after the DIV element.

    Block

    So far, we talked about inline. Now let's switched to the opposite of it, block. Remember inline elements appears on the same line. Well, block starts on a NEW line and takes up the full width available. So that means block elements will occupy the entire width of its parent element. #block elements
    Here are a few elements that have a default block property:

    div
    h1
    p
    li
    section
